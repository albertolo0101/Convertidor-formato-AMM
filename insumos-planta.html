⚙ Configuración requerida

Abrí este archivo en un editor de texto y completá el bloque CONFIG al inicio del <script>:

  1. Ir a console.cloud.google.com
  2. Habilitar la Google Sheets API
  3. Crear una API Key → restringir a Sheets API → pegar en API_KEY
  4. Crear un OAuth 2.0 Client ID (tipo: Web application) → Authorized JavaScript origins: agregar null (para archivos locales) → pegar en CLIENT_ID
  5. Abrir el Google Sheet → copiar el ID del URL (entre /d/ y /edit) → pegar en SHEET_ID
  6. Compartir el Sheet con tu Gmail personal como Editor

⚠ Las escrituras requieren OAuth. El CLIENT_ID es necesario para agregar o editar items.
